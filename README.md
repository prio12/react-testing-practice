# React Testing Practice

A practice repository for learning **React component testing** using **Vite**, **TypeScript**, **Vitest**, **React Testing Library**, and **jest-dom**.  
This project demonstrates setting up a fully working testing environment and writing basic tests for React components.

---

## Features Practiced

- Vite + React + TypeScript setup  
- Vitest as the test runner  
- React Testing Library for user-centric UI testing  
- jest-dom for extended matchers like `toBeInTheDocument()`  
- Writing basic unit tests for React components
